# coc-anvil

Anvil Language Server coc.nvim extension

## Install

`:CocInstall coc-anvil`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-anvil-keymap)`

## Lists

`:CocList demo_list`
